# 51单片机学习笔记

## 一、51单片机总体架构认知

51单片机本质上是一颗 **微型计算机**，内部包含五大核心模块：

CPU（运算核心）
RAM（数据存储）
ROM（程序存储）
IO口（控制外设）
定时器/串口（功能模块）

## 二、引脚功能总览

标准51单片机共 **40个引脚**，分为四类：

| 类型         | 作用           |
| ------------ | -------------- |
| 电源引脚     | 供电           |
| 控制引脚     | 复位、时钟     |
| IO引脚       | 控制外设       |
| 特殊功能引脚 | 串口、外部中断 |

## 三、P0–P3口

51最重要的就是 **四个IO端口**P1P2P3P0

每个端口都是 **8位**

所以总IO数：8 × 4 = 32个IO口

### 四、P0–P3口区别

| 端口 | 特点     | 是否有特殊功能 | 上拉电阻 |
| ---- | -------- | -------------- | -------- |
| P0   | 最特殊   | 外部总线       | 无       |
| P1   | 最简单   | 无             | 有       |
| P2   | 地址高位 | 外扩存储       | 有       |
| P3   | 功能最多 | 串口/中断      | 有       |

## 五、P0口

特点：

开漏结构
必须外接上拉电阻才能输出高电平

用途：

① 普通IO
② 地址/数据总线

> P0口输出1 = 释放总线
> P0口输出0 = 拉低电平

## 六、P1口

- 内部自带上拉
- 没有复用功能
- 最适合控制LED、数码管

## 七、P2口

作用：外扩存储器时 提供高8位地址

普通情况下：当普通IO使用

## 八、P3口

P3每一位都有特殊功能：

| 引脚 | 功能        |
| ---- | ----------- |
| P3.0 | 串口RX      |
| P3.1 | 串口TX      |
| P3.2 | 外部中断0   |
| P3.3 | 外部中断1   |
| P3.4 | 定时器0输入 |
| P3.5 | 定时器1输入 |
| P3.6 | 外部写信号  |
| P3.7 | 外部读信号  |

# 十、核心寄存器基础认知

寄存器本质：

> 单片机内部的变量

直接控制硬件

## 最重要寄存器（必须认识）

### ① P0–P3寄存器

控制IO口输出电平

### ② TMOD（定时器模式寄存器）

控制定时器工作模式

### ③ TCON（定时器控制寄存器）

控制定时器启动停止

### ④ SCON（串口控制寄存器）

控制串口通信方式
