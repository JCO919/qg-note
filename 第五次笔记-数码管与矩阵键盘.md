# 静态数码管原理

## 一、数码管结构

数码管由 **7个LED段 + 1个小数点** 组成。

每个段对应一个 LED，通过控制亮灭组合显示不同数字。

## 二、共阴与共阳

### 1 共阴极数码管

所有 LED 阴极接地。

特点：

- 段选 **高电平点亮**
- 段选 **低电平熄灭**

### 2 共阳极数码管

所有 LED 阳极接 VCC。

特点：

- 段选 **低电平点亮**
- 段选 **高电平熄灭

------

## 三、数码管编码表

编码表用于存储 **数字对应的段码**。

例如：

```
unsigned char code seg_code[]={
0x3f, //0
0x06, //1
0x5b, //2
0x4f, //3
0x66, //4
0x6d, //5
0x7d, //6
0x07, //7
0x7f, //8
0x6f  //9
};
```

说明：

```
bit0 -> a
bit1 -> b
bit2 -> c
bit3 -> d
bit4 -> e
bit5 -> f
bit6 -> g
bit7 -> dp
```

#include <reg52.h>

unsigned char code seg_code[]={

# 数码管动态显示

## 一、为什么需要动态显示

开发板通常有 **多个数码管**。

如果全部同时点亮：

- IO 不够
- 成本高

解决方法：

**动态扫描**

------

## 二、动态扫描原理

通过 **快速轮流点亮每个数码管**。

流程：

```
点亮第1位 → 关闭
点亮第2位 → 关闭
点亮第3位 → 关闭
...
```

扫描速度足够快，人眼看起来像 **同时亮**。

------

## 三、位选与段选

### 段选

控制 **显示数字**

例如：

```
P0 -> 段码
```

### 位选

控制 **第几个数码管亮**

例如：

```
P2 -> 位选
```

# 实验：0-9计数

### 实验目标

每秒增加1

------

### 实验代码

```
#include <reg52.h>

unsigned char code seg_code[]={
0x3f,0x06,0x5b,0x4f,0x66,
0x6d,0x7d,0x07,0x7f,0x6f
};

void delay(unsigned int t)
{
    unsigned int i,j;
    for(i=0;i<t;i++)
        for(j=0;j<120;j++);
}

void main()
{
    unsigned char num=0;

    while(1)
    {
        P0 = seg_code[num];
        delay(1000);

        num++;

        if(num>9)
            num=0;
    }
}
```

# 矩阵键盘原理

## 一、为什么使用矩阵键盘

如果使用独立按键：

```
16个按键 = 16个IO
```

矩阵键盘只需要：

```
4行 + 4列 = 8个IO
```

------

## 二、矩阵结构

```
    C1 C2 C3 C4
R1  1  2  3  A
R2  4  5  6  B
R3  7  8  9  C
R4  *  0  #  D
```

------

## 三、扫描原理

步骤：

1. 行全部置高
2. 逐行置低
3. 读取列状态
4. 判断按键位置

示例：

```
R1 = 0
检测 C1 C2 C3 C4
```

如果 C2=0

说明按键：

```
R1 C2
```

# 实验：矩阵键盘扫描

### 实验代码

```
#include <reg52.h>

unsigned char key_scan()
{
    unsigned char key=0;

    P1 = 0x0f;

    if(P1 != 0x0f)
    {
        delay(10);

        if(P1 != 0x0f)
        {
            switch(P1)
            {
                case 0x0e: key=1; break;
                case 0x0d: key=2; break;
                case 0x0b: key=3; break;
                case 0x07: key=4; break;
            }
        }
    }

    return key;
}
```
